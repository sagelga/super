# ByteSide Retriever

Cloudflare Worker that retrieves content from Notion and converts it to Markdown for ByteSide.

## Stack

| Layer | Technology |
|---|---|
| Runtime | Cloudflare Workers |
| Source | Notion API |
| Conversion | notion-to-md |
| Language | TypeScript |
