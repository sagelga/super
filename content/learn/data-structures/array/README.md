# Arrays

อาร์เรย์เป็นโครงสร้างข้อมูลพื้นฐานที่เก็บข้อมูลในตำแหน่งหน่วยความจำติดกัน

## คุณสมบัติ

- **เข้าถึงสุ่ม**: สามารถเข้าถึงองค์ประกอบใดก็ได้ใน O(1)
- **ขนาดคงที่**: ต้องกำหนดขนาดตอนสร้าง (ในภาษาบางภาษา)
- **หน่วยความจำติดกัน**: องค์ประกอบเก็บในหน่วยความจำที่ติดกัน

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access    | O(1)       |
| Search    | O(n)       |
| Insert    | O(n)       |
| Delete    | O(n)       |

## ตัวอย่าง

```python
# Python
arr = [1, 2, 3, 4, 5]
print(arr[2])  # 3

# เพิ่มองค์ประกอบ
arr.append(6)  # [1, 2, 3, 4, 5, 6]
```

```javascript
// JavaScript
const arr = [1, 2, 3, 4, 5];
console.log(arr[2]); // 3

// เพิ่มองค์ประกอบ
arr.push(6); // [1, 2, 3, 4, 5, 6]
```
